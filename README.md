# 🏥 SaludYa - Frontend (Fisioterapia)

Aplicación web desarrollada en Angular para la gestión de citas en consultorios de fisioterapia.

## 🚀 Descripción

Este proyecto corresponde al frontend del sistema **SaludYa**, enfocado en la gestión de servicios de fisioterapia y rehabilitación.

El sistema permite a los pacientes agendar citas según el siguiente flujo:

1. Seleccionar especialidad (ej: Fisioterapia)
2. Seleccionar categoría de servicio
3. Seleccionar servicio específico
4. Elegir profesional disponible
5. Agendar cita según horarios disponibles

---

## 🧱 Arquitectura

El proyecto sigue una estructura organizada por capas

---

## 🔐 Características principales

- Autenticación por roles
- Selección de servicios de fisioterapia
- Clasificación por categorías:
  - Evaluación inicial
  - Terapias físicas
  - Terapias con equipos
  - Rehabilitación específica
- Visualización de profesionales disponibles
- Gestión de citas
- Visualización de horarios

---

## 🎯 Objetivo

Proveer una interfaz moderna para la gestión de servicios de fisioterapia, facilitando la programación de citas y el acceso a tratamientos especializados.
